# KineGuard 세션 요약 — 2026년 4월 1일

## 프로젝트 개요

KineGuard는 스켈레톤 기반 영상 콘텐츠 안전 분류기로, 4단계 서열 척도를 사용한다:
- **Tier 0:** 일반 — 일상적 비댄스 동작 (걷기, 앉기, 손 흔들기)
- **Tier 1:** 예술적 — 고에너지 건전 댄스/운동 (K-pop, 브레이킹, 힙합 안무)
- **Tier 2:** 선정적 — 관능적/성적 암시 동작 (트워킹, 플로어워크, 골반 롤링, 체어댄스)
- **Tier 3:** 노골적 — 포르노그래피 또는 명백한 성적 행위

**핵심 설계 원칙:** KineGuard는 외모가 아닌 **동작 패턴**을 분류한다. 비키니든 트레이닝복이든 동일한 힙 아이솔레이션은 동일한 tier를 받아야 한다. 스켈레톤 표현은 외모 편향(피부, 의상, 카메라 앵글)을 제거한다.

**파이프라인:** 영상 → WHAM (3D 스켈레톤 추출, 31 Halpe 관절) → LMA 특성 추출 (프레임당 55개 특성, Laban Movement Analysis) → 분류기 (미학습)

## 시도한 것과 실패한 것

### 1. DEMO (Dense Motion Captioning) — 실패
- LLaMA-3.1-8B + MLP 모션 어댑터, AMASS/HumanML3D로 학습
- SMPL 22관절 스켈레톤 입력 → 텍스트 캡션 생성
- **결과:** 모든 댄스 영상이 "군대식 포복 전진", "수영 동작", "물구나무서기"로 캡션됨. 8개 중 7개 영상에서 Tier 분류 불가(N/A).
- **원인:** 학습 데이터(AMASS/HumanML3D)에 댄스 콘텐츠가 전무. 트워킹을 가장 유사한 학습 예제(앞으로 구부림 = 수영)로 매핑.
- `eval/kineguard_baseline.py` — 이 실패를 재현하는 스크립트 구축.

### 2. Qwen3-VL-Embedding-2B (원본 영상) — 작동하지만 편향됨
- 비전-언어 임베딩 모델, tier 텍스트 프롬프트 대비 제로샷 코사인 유사도
- **결과:** 체어댄스를 tier2로 정확히 분류 (점수 0.579). 8개 테스트 영상 모두 작동.
- **문제:** 시각적 외모 기반 분류 — 피부 노출, 의상, 카메라 앵글. 다른 옷을 입은 동일한 댄스에 다른 점수 부여. KineGuard가 제거하려는 바로 그 편향.

### 3. Qwen3-VL (렌더링된 스켈레톤 영상) — 실패
- 동일 모델에 WHAM 출력으로 렌더링한 스틱 피겨 스켈레톤 영상 입력
- **16개 영상 테스트:** Tier 2 (선정적) 8개 + Tier 1 (예술적) 8개
- **16개 영상 모두 실제 tier와 무관하게 tier1로 분류**
- **T2 평균 tier2 점수: 0.406 vs T1 평균 tier2 점수: 0.404 — 차이 +0.002 (분리 없음)**
- **결론:** 스켈레톤 영상에 대한 임베딩 유사도로는 선정적 댄스와 예술적 댄스를 구분할 수 없음.

### 4. MotionScript (규칙 기반 스켈레톤→텍스트) — 부분적 성공
- 3D 스켈레톤을 구조적 자연어로 변환하는 규칙 기반 시스템 (IROS 2025)
- **MotionScript → Claude (블라인드 추론 LLM):** 트워크 영상을 **높은 확신도로** Tier 2로 정확히 분류
  - 몸통 반복적 수평, 손이 몸 뒤/엉덩이 아래, 리듬적 힙 진동 패턴
- **MotionScript → Qwen3 임베딩 유사도:** 실패. tier1(0.653) > tier2(0.638).
- **결론:** 기하학적 신호는 스켈레톤에 존재하지만, 임베딩 분류기가 아닌 추론 LLM만이 추출 가능.

### 5. 모션 LLM 조사 — 기존 모델로는 해결 불가
조사 모델: MotionLLM, M3GPT, MG-MotionLLM, MotionGPT, LMM, Being-M0, MotionScript, Superman, ViMoNet.

**결론:** 어떤 공개 모델도 "선정적" vs "예술적" 구분을 학습 어휘에 포함하지 않음. 이것이 KineGuard가 채우는 진정한 연구 공백.

## 실제로 작동하는 것: LMA 특성 분석

### 돌파구
WHAM 파이프라인에서 이미 LMA 특성 55개를 추출 중 (Body/Effort/Shape/Space). Turab et al. (arXiv:2504.21166) 논문은 동일 LMA 특성 + SVM/RF로 AIST++에서 댄스 스타일 분류 **99.18% 정확도** 달성.

### Gemini Deep Research 분류 체계
**예측된 Tier 1 (운동적):** Bound Flow, Strong Weight, Sudden Time, Direct Space
**예측된 Tier 2 (선정적):** Free Flow, Light Weight, Sustained Time, Indirect Space

### 핵심 발견

#### 1. 세 Tier 전반에 걸친 단조 기울기 (MONOTONIC GRADIENT)
거의 모든 특성이 깨끗한 서열 기울기 형성: T1 → T2 → T3. Tier 2는 별도 범주가 아닌, 운동적 동작과 노골적 동작 사이 연속체의 중간 상태.

| 신호 | T1 → T2 → T3 | 패턴 |
|---|---|---|
| 궤적 경로 길이 | 4.12 → 2.21 → 0.74 | 전신 움직임 점감 |
| 골반 운동에너지 변동성 | 0.55 → 0.84 → 1.84 | 골반 점점 불규칙 |
| 골반 직접성 | 28.0 → 19.4 → 8.9 | 골반 점점 간접적 |
| Effort Time | 297 → 207 → 90 | 더 지속적, 덜 급격 |
| Flow CV | 0.35 → 0.47 → 0.60 | Bound-free 진동 증가 |

#### 2. 팔 비활동성이 T1 vs T2의 가장 강한 신호 (d > 1.0)
상위 7개 분리 특성 모두 팔/손목의 비활동성:
- 손목 가속도: T2가 47% 낮음 (d=1.37)
- 손목 운동에너지: T2가 61% 낮음 (d=1.21)

선정적 댄스는 팔을 내리거나, 뒤로 하거나, 정지. 예술적 댄스는 복잡한 궤적의 활발한 팔 사용.

#### 3. 골반 불규칙성 + 전반적 정지가 T2 vs T3 신호 (d > 1.0)
- 골반 직접성: d=1.59 (T3 훨씬 간접적)
- 골반 운동에너지 변동성: d=1.54 (T3 훨씬 불규칙)
- 궤적 경로 길이: d=1.52 (T3 거의 정지)

T3는 T2의 증폭 버전: 더 적은 전신 움직임, 더 불규칙한 골반, 더 간접적이고 지속적.

#### 4. T1 vs T3 효과 크기 매우 큼 (d = 2.0-2.9)
- Effort Time: d=2.94, 골반 저크: d=2.86, Effort Flow: d=2.56, 골반 속도: d=2.52

#### 5. Gemini 분류 체계 검증
- **간접적 골반 → 확인** (직접성: T1=28.0, T2=19.4, T3=8.9)
- **자유로운/가변적 flow → 확인** (Flow CV: T1=0.35, T2=0.47, T3=0.60)
- **가벼운 weight → 확인** (Effort Weight: T1=37.6, T2=25.2, T3=8.1)
- **지속적 time → 확인** (Effort Time: T1=297, T2=207, T3=90)
- **Gemini에 없던 새 발견:** 팔 비활동성이 T1과 T2 간 #1 차별화 요소

### LMA 서명 요약
- **T1 (예술적) = 활발한 팔 + 구조화된 직접적 전신 동작 + 강하고 급격한 effort**
- **T2 (선정적) = 조용한 팔 + 불규칙 간접적 골반 + 중간 flow 변동성**
- **T3 (노골적) = 거의 정지한 몸 + 최대 불규칙 간접적 골반 + 지속적 가벼운 flow**

## 향후 과제

1. **LMA 특성으로 분류기 학습 (SVM/RF).** Cohen's d 값(T1-T2: >1.0, T1-T3: >2.0)은 가능성을 강하게 시사.

2. **데이터셋 확장.** Tier당 100개 이상 필요. 현재 T3 1000개 처리 중 (NPDI, 3 GPU, ~1일 소요).

3. **서열 기울기가 핵심 논문 인사이트.** 서열 회귀 모델이 표준 다클래스 분류기를 능가할 수 있음.

4. **OMIL (Ordinal Multiple Instance Learning):** 웹 크롤링 잡음 라벨 처리(MIL) + 서열 tier 기울기 존중(ordinal).

5. **논문 기여:**
   - 기여 1: 기존 모델이 스켈레톤 기반 NSFW 분류를 처리하지 못함을 증명
   - 기여 2: 3-tier LMA 기울기 발견 (팔 비활동성 + 골반 불규칙성 + flow 변동성)
   - 기여 3: KineGuard — 학습된 스켈레톤 분류기 자체
   - 기여 4: VLM 기반 콘텐츠 조절의 외모 편향 문제
